3. **View online**

## 🌐 GitHub Pages Deployment

### Setup Instructions

1. **Push to GitHub**
   ```bash
   git init
   git add .
   git commit -m "Initial commit: Gitanjali Couture landing page"
   git branch -M main
   git remote add origin https://github.com/yourusername/GitanjaliCouture.git
   git push -u origin main
   ```

2. **Enable GitHub Pages**
   - Go to your repository on GitHub
   - Navigate to Settings → Pages
   - Under "Source", select `main` branch
   - Click Save
   - Your site will be published at `https://yourusername.github.io/GitanjaliCouture/`

3. **Alternative: Make it your user site**
   - Rename the repository to `yourusername.github.io`
   - Your site will be available at `https://yourusername.github.io/`

## 🔧 Customization

### Update Contact Information
Edit the contact details in `index.html`:
```html
<p class="phone">
    <span class="phone-icon">☎</span>
    +91 YOUR_PHONE_NUMBER
</p>
```

### Update Social Media Links
Update the social media URLs in `index.html`:
```html
<a href="https://www.instagram.com/YOUR_HANDLE" target="_blank" class="social-link instagram">
```

### Modify Colors
Update color variables in `styles.css`:
- Primary Gold: `#d4af37`
- Dark Background: `#140a05`
- Light Background: `#f5f0ea`
